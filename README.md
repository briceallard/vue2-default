#vue2-default

Default Vue2 Template with Vuex, Vuetify and v-router installed via NPM

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
